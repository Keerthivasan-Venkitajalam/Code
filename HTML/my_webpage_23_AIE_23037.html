Project experiences of SEMESTER 1

updated 18th february, 2024...

I am thrilled to announce that I have completed my first semester of my UG course on Computer science in a specialization in AI, It has been a journey of learning and growth, and I am proud of the projects I have accomplished in each of my subjects:

Mathematics for Computing: I developed a facial recognition system using Principal Component Analysis (PCA) and Singular Value Decomposition (SVD). This project allowed me to apply mathematical concepts to real-world applications.

Problem Solving & C Programming: I created a header file for matrix operations in C programming. This project helped me understand the importance of efficient code organization and reusable components in software development.

Elements of Computing: I designed a CPU architecture and implemented a serial binary adder using SR flip-flops in Hardware Description Language (HDL). This project gave me a deeper understanding of how computers work at the hardware level.

Computational Physics: I worked on several projects related to air flight dynamics. These included the analysis and optimization of aircraft fuel consumption and emissions, the conceptualization of future aircraft, and the creation of a mathematical model for fuel consumption. These projects allowed me to apply physics concepts to practical problems in aviation.

Discrete Mathematics: I developed an email spam classifier using the Naive Bayes classifier. This project allowed me to apply mathematical concepts to machine learning and data classification. I used a corpus of spam and ham emails to train and test the classifier. It worked well but yet a primitive model indeed.

I am grateful for the knowledge I have gained and look forward to applying these skills in my future endeavors. Here’s to more learning and growth in the coming semesters!
